# Superstore Sales Dashboard (Power BI)

This project is a sales analysis dashboard built using Power BI. The goal of this project was to explore the Superstore dataset and understand sales performance, profit trends, and customer segments through visual analysis.

## Project Overview

The dashboard provides insights into:
- Sales performance across different product categories
- Performance of various sub-categories
- Monthly sales and profit trends
- Regional distribution of sales
- Customer segments contributing to revenue

The dashboard is interactive and allows filtering by region to better understand sales patterns.

## Dashboard Insights

Some key observations from the dashboard:

- Office Supplies generates the highest sales among all categories.
- Phones and Tables are among the top-performing sub-categories.
- The Consumer segment contributes the largest portion of sales.
- Sales tend to increase toward the end of the year.
- Certain regions show higher sales concentration.
